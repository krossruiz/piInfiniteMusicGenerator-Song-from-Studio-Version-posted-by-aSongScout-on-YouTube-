How It Works

Each digit of π is mapped to a note on the A Harmonic Minor scale, creating an endless, never-repeating melody. The left hand plays arpeggiated accompaniment derived from the harmony of each measure. Since π is irrational, the music never loops and never ends.

Digits are computed live using an unbounded spigot algorithm — no lookup table, no limit. This page will generate new digits of π forever.

0 → A3 1 → B3 2 → C4 3 → D4 4 → E4 5 → F4 6 → G#4 7 → A4 8 → B4 9 → C5
